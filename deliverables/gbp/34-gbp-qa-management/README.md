# Retired GBP Q&A Workflow

**Category:** GBP SEO
**Automation Readiness Score:** 0/10 - Retired / no execution
**Status:** Workflow removed

---

## Purpose

Process 34 previously covered Google Business Profile Q&A management. That workflow is now retired because Google Business Profile no longer supports or uses the Q&A feature as an active profile-management surface.

This SOP exists only to prevent the old task from being reintroduced into GBP fulfillment.

## Current State

Do not run a GBP Q&A workflow.

Do not:

- Research, draft, seed, publish, answer, monitor, or report on GBP Q&A.
- Use the old `Q&As Submitted` ClickUp task as an active fulfillment step.
- Use the former GBP Q&A Generator Assistant for production work.
- Ask operators to check Google Maps or GBP for Q&A availability.
- Route review, service, photo, post, or profile findings into Process 34.
- Count Q&A work as part of GBP profile completion.

## Automation Score

**0/10 - Retired / no execution**

No automation should be built for this process. The correct automation status is `0/10` because the process is obsolete, not because it is manual.

## Replacement Routing

When customer questions, service confusion, or repeated review themes come up, route them to active workflows instead:

| Need | Active Workflow |
|---|---|
| Website FAQ or service-page clarification | Process 02 - Service Pages |
| Blog or support article opportunity | Process 04 - Blog Content |
| GBP post topic | Process 31 - Weekly GBP Posts |
| Review reply or reputation issue | Process 32 - Automated Review Replies |
| Service/category mismatch | Process 30 - GBP Categories |
| Profile facts, services, products, or attributes issue | Process 29 / Process 36 |

## Source Notes

The legacy ClickUp task `Q&As Submitted` and the prior GBP Q&A references are retained only as historical context. They are not active SOP instructions.

## QA Checklist

- [ ] No GBP Q&A research was assigned.
- [ ] No GBP Q&A drafting was assigned.
- [ ] No GBP Q&A posting or answering was assigned.
- [ ] No GBP Q&A monitoring was assigned.
- [ ] Any customer-question insight was routed to an active workflow instead.
- [ ] ClickUp or tracker notes mark the old Q&A task as obsolete when encountered.

## Completion Criteria

Process 34 is complete when the obsolete Q&A workflow is skipped and any useful customer-question insight has been routed to a current content, review, post, service, or profile workflow.
